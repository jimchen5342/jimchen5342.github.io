偽類別選擇器一(pseudo class)

  1. :link

    ：選擇未被訪過的連結 
            
          a:link { }

  2. :visited

    ：選擇已被訪過的連結 
            
          a:visited { }

  3. :hover

    ：選擇滑鼠游標停留的元素 
            
          div:hover { }

  4. :focus

    ：選擇獲得焦點的元素 
            
          input:focus { }

  5. :enabled

    ：選擇啓用的 input 元素 
            
          input:enabled { }

  6. :disabled

    ：選擇禁用的 input 元素 
            
          input:disabled { }

  7. :checked

    ：選擇被選中的 input 元素 
            
          input:checked { }
