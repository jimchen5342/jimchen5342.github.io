簡單選擇器

  1. *：萬用字元 
            
          * {
          }

  2. tagName：

            
          <div> </div>
        
        
          div{
          }

  3. #id：

            
          <div id="container"> </div>
        
        
          #container {
          }

  4. .className：

            
          <div class="container"> </div>
        
        
          .container {
          }

  5. tagName.className

            
          <div class="container"> </div>
        
        
          div.container {
          }
