屬性選擇器

  1. [attribute]：選擇帶有 attribute 的所有元素 
            
          <a target='_blank'></div>
        
        
          [target] {
          }

          a[target] {
          }
  2. [attribute=value]：選擇 attribute 帶有 value 的所有元素 
            
          <a target='_blank'></div>
        
        
          [target=_blank] {
          }
  3. [attribute~=value]：選擇 attribute 包含 value(單字) 的所有元素 
            
          <a target='_parent _blank'></div>
        
        
          [target~=_blank] {
          }
    4. 註：HTML5 允許元素可以自訂 attribute
