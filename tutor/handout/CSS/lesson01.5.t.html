偽類別選擇器二(pseudo class)

  1. :first-child

    ：父元素內部指定元素的第一個 
            
          ul > li:first-child {
          }

  2. :last-child

    ：父元素內部指定元素的最後一個 
            
          ul > li:last-child {
          }

  3. nth-child(n)

    ：父元素內部指定元素，可以指定奇、偶數或一個由 1 開始的數字 (參數：odd|even|number) 
            
          ul > li:nth-child(odd) {
          }
          ul > li:nth-child(even) {  
          }
          ul > li:nth-child(1) {
          }

  4. nth-last-child(n)

    ：父元素內部指定倒數的元素(參數：number，從 1 開始) 
            
          ul > li:nth-last-child(1) {
          }
