偽元素選擇器(pseudo element)

  1. :first-line

    ：選取第一行 
              
            div:first-line {
              color: red;
            }

  2. :first-letter

    ：選取第一個字 
              
              div:first-letter {
                color: red;
              }

  3. :before

    ：從選取元素的前面插入元素 
              
            div:before {
              content: "";
              color: red;
            }

  4. :after

    ：從選取元素的後面插入元素 
              
              div:after {
                content: "";
                color: red;
              }

  5. :selection

    ：選取文字反白後 
              
              div:selection {
                color: red;
              }
