CSS 用於控制網頁的樣式和布局

用法

  1. 內聯：寫在元素的 style 屬性 
    
            <input type="button" style="color: red;" />
  2. 外部/內部： 
    
            body {
              color: red;
            }

注意事項

  1. inherit
  2. !important
  3. CSS 屬性前綴（Vendor Prefixes）
