CSS 用於控制網頁的樣式和布局

用法

  1. 屬性名稱: 屬性值;
  2. 內聯：寫在元素的 style 屬性 
    
            <input type="button" style="color: red; font-size: 16px;" />
  3. 外部/內部：利用 Selector(選擇器)指定元素套用條件，並用 { } 含括屬性 
    
            input {
              color: red;
              font-size: 16px;
            }

注意事項

  1. inherit
  2. !important
  3. none
  4. CSS 屬性前綴（Vendor Prefixes） 
    
          .box {
            -moz-border-radius: 10px;
            -ms-border-radius: 10px;
            -o-border-radius: 10px;
            -webkit-border-radius: 10px;
            border-radius: 10px;
          }
