文本

  1. color

    ：
    • 顏色名稱：red, green, blue
    • 十六進位：#ff0000, #00ff00, #0000ff
    • 十進位函式：rgb(255, 0, 0), rgb(0, 255, 0), rgb(0, 0, 255)
    • 十進位函式：rgba(255, 0, 0, 0.5), rgba(0, 255, 0, 0.5), rgba(0, 0, 255, 0.5)
      opacity：0~1
    • hsla
            
          div {
            color: rgb(0, 255, 0);
          }

  2. text-aling

    ：left(default) | right | center 
            
          div {
            text-aling: right;
          }

  3. vertical-align

    ：top | text-top | middle(default) | bottom | text-bottom
    用於表格 
            
            td {
              vertical-aling: top;
            }

  4. text-overflow

    ：ellipsis ｜clip
            
          div {
            text-overflow: ellipsis;
            white-space : nowrap;
            overflow: hidden;
          }

  5. white-space

    ：normal | nowrap 
            
            div {
              white-space : nowrap;
            }

  6. word-spacing

    ：normal(單字間距 25px) | px | em 
            
            div {
              word-space : 40px;
            }

  7. letter-spacing

    ：normal(字元間距 0px) | px | em 
            
            div {
              word-space : 5px;
            }

  8. text-transform

    ：none | capitalize | uppercase | lowercase 
            
          div {
            text-transform: capitalize;
          }

  9. text-decoration

    ：none | underline | overline | line-through 
            
          div {
            text-decoration: underline;
          }

  10. text-shadow

    ：水平距離 垂直距離 模糊距離 陰影顏色 
            
          div {
            text-shadow: 5px 5px 5px #FF0000;
          }
