背景

  1. background

    ： 
              
              div {
                background: red;
              }
              div {
                background: #00FF00 url(bgimage.gif) no-repeat top;
              }
              div {
                background: -webkit-linear-gradient(top,  #f2f5f6, #c8d7dc);
              }

  2. background-color

    ： 
              
            div {
              background-color: red;
            }

  3. background-image

    ：背景圖片

    • background-image

      ：url("./images/bg.jpg")

    • background-position

      ： 
                      
                  div {
                    background-position: center;
                  }
                  div {
                    background-position: left top;
                  }
                  div {
                    background-position: 50%;
                  }
                  div {
                    background-position: 20% 50%;
                  }

    • background-repeat

      ：repeat(default) | repeat-x | reapeat-y | no-repeat

    • background-size

      ：

      • auto

        ：依原圖尺寸，default

      • %

        ：百分比

      • length

        ：px

      • cover

        ：背景圖放大至內容的大小

      • contain

        ：背景圖縮小至內容的大小
      註：length 與 % 可設定 2 個數值，也可只設定 1 個數值， 當只設定一個數值，此時背景圖原始寬高比例，自動縮放

  4. background-image

    ：漸層 
              
            div { 
               background-image: -webkit-linear-gradient(red, blue); 
            }
            div {
              background-image: -webkit-linear-gradient(top,  #f2f5f6, #c8d7dc);
            }
            div {
              background-image: -webkit-gradient(linear, right top, left bottom, 
                color-stop(0, #FFFFFF), color-stop(100, #3366FF));
            }
