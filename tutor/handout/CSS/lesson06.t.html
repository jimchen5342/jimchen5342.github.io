定位

  1. position

    ：

    • static

      ：依瀏覽器預設的配置排版，座標屬性不會作用；預設值

    • relative

      ：依瀏覽器預設的配置排版，如有設定座標屬性，將會依其設定作偏移

    • absolute

      ：依父元素(要設 relative)或 body 為參考點；並設定座標屬性，作偏移

    • fixed

      ：依 window 為參考點；並設定座標屬性，作偏移
              
            div {
              position: absolute;
              left: 50%;
            }

  2. 座標屬性

    ：

    • top

      ：% | px

    • bottom

      ：% | px

    • left

      ：% | px

    • right

      ：% | px

  3. z-index

    ：重疊順序，預設為 0 
              
            div {
              z-index: 10;
            }
