布局

  1. display

    ：

    • none

      ：隱藏元素，不保留其實際空間

    • inline

      ：內聯元素，無法設定 width 和 height
      span, a 即預設為該屬性

    • block

      ：區塊元素，預設換行
      h1~h6, p, div 即預設為該屬性

    • inline-block

      ：內聯區塊元素，不換行，可設定 width 和 height

    • table | table-cell | table-row

      ：表格元素 
              
            div {
              display: none;
            }
    註：在 Flex box 說明下列屬性： box | inline-box | flexbox | inline-flexbox | flex | inline-flex

  2. visibility

    ：

    • visible

      ：顯示元素

    • hidden

      ：隱藏元素，保留其實際空間
              
            div {
              visibility: hidden;
            }

  3. overflow

    ：元素內容超出；visible | hidden | scroll | auto

    • overflow-x

      ：元素內容超出寬度；visible | hidden | scroll | auto

    • overflow-y

      ：元素內容超出高度；visible | hidden | scroll | auto
              
            div {
              height: 100px;
              overflow: hidden;
            }

  4. float

    ：left | right 
              
            div {
              float: left;
            }

  5. clear

    ：none | both | left | right 
            
          div {
            clear: none;
          }
