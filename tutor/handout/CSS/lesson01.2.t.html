複合選擇器

  1. selector1, selector2：selector1 和 selector2...... 
            
          <div class='div1'></div>
          <div class='div2'></div>
          <div id='div3'></div>
        
        
          .div1, .div2, #div3 { }
  2. selector1 selector2：selector1 內部所包含的所有後裔, selector2 元素 
            
          <div> 
            <span> 
              <a>anchor</a> 
            </span> 
          </div>
        
        
          div a { }
  3. selector1 > selector2：selector1 內部所包含的第一層後裔, selector2 元素 
            
          <div id="container">
            <a>anchor</a>
          </div>
        
        
          #container > a { }
  4. selector1 + selector2：selector1 之後相鄰的『第一個』selector2 元素 
            
          <div></div>
          <p>p1</p>
          <p>p2</p>
        
        
          div + p { }
註：注意空白的用法
