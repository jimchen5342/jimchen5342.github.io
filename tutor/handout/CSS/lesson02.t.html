字體

  1. font

    ：font-style | font-weight | font-size/line-height | font-family 
            
          div {
            font: italic border 1em/20px Arial;
          }

  2. font-style

    ：normal(default) | italic(斜體字) | oblique(文字傾斜) 
            
          div {
            font-style: italic;
          }

  3. font-weight

    ：粗體字
    • 100 ~ 900
    • lighter = 100
    • normal = 400
    • bold = 700
    • bolder = 900
            
          div {
            font-weight: bold;
          }

  4. font-size

    ：尺寸(px | em | rem) 
            
          div {
            font-size: 16px;
          }

  5. line-height

    ：字體的行高(px | em | rem) 
            
          div {
            line-height: 24px;
          }

  6. font-family

    ：字體名稱 
            
          div {
            font-family: 'Arial Black', 微軟正黑體, Helvetica, Arial;
          }

單位

  1. %

    ：百分比

  2. in

    ：英吋

  3. cm

    ：公分

  4. mm

    ：亳米

  5. pt

    ：1pt = 1/72 in

  6. px

    ：畫素，網頁文字尺寸預設為 16畫素

  7. em

    ：是相對於外圍的文字大小

  8. rem

    ：僅相對於 root 層級的文字大小
