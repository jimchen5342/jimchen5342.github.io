Flex Box


圖片來源：MDN
  1. 父元素(container)

    • display

      ：flex | inline-flex

    • flex-direction

      ：

      • column

        ：從上到下(default)

      • row

        ：從左到右
        • 註：row-reverse, column-reverse

    • flex-wrap

      ：nowrap(default) | wrap | wrap-reverse

    • flex-flow

      ：flex-direction flex-wrap

    • justify-content

      ：決定 flex 項目與 main axis 的對齊設定

      • flex-start

        ：項目往 main axis 的 main start 對齊(default)

      • flex-end

        ：項目 main axis 的 main end 對齊

      • center

        ：置中

      • space-between

        ：平均分配內容元素，左右元素將會分別與 main axis 的 main start 和 main end 貼齊

      • space-around

        ：平均分配內容元素，間距也是平均分配

    • align-items

      ：決定 flex 項目與 cross axis 的對齊設定

      • flex-start

        ：項目往 cross axis 的 cross start 對齊(default)

      • flex-end

        ：項目往 cross axis 的 cross end 對齊

      • center

        ：置中

      • stretch

        ：將內容元素全部撐開填滿

      • baseline

        ：以所有內容元素的基線作為對齊標準

    • align-content

      ：功能同 align-items，但是是針對多列元素

      • flex-start

        ：項目往 cross axis 的 cross start 對齊(default)

      • flex-end

        ：項目往 cross axis 的 cross end 對齊

      • center

        ：置中

      • stretch

        ：自動將多餘的空間補滿(default)

      • space-between

        ：將空白平均分配到每行內容之間，第一行靠在 cross-start，最後一行靠在 cross-end

      • space-around

        ：平均分配空白到每行內容之間
  2. 子元素(item)

    • align-self

      ：屬性同 align-item；但是，針對子元素覆寫父元素的 align-item

    • flex-grow

      ：number(預設 0)，按照比例來填滿剩餘的空間

    • flex-shrink

      ：number(預設 1)，縮小成設定的比例

    • flex-basis

      ：px | % | auto(default)，定義的最小寬度或高度

    • flex

      ：none | flex-grow flex-shrink flex-basis，預設值為 0 1 auto
下列屬性值是不同階段所定義的結果：
box | inline-box | flexbox | inline-flexbox | flex | inline-flex
