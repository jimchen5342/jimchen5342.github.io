Flex Box

  1. 父元素(container)

    • display

      ：flex | inline-flex

    • flex-direction

      ：

      • column

        ：從上到下(default)

      • row

        ：從左到右
        • 註: row-reverse, column-reverse

    • justify-content

      ：決定 flex 項目與容器的水平對齊設定

      • flex-start

        ：對齊容器最左邊(default)

      • flex-end

        ：對齊容器最右邊

      • center

        ：水平置中

      • space-between

        ：平均分配內容元素，左右元素將會與容器的最左邊和最右邊貼齊

      • space-around

        ：平均分配內容元素，間距也是平均分配

    • align-items

      ：決定 flex 項目與容器的垂直對齊設定

      • flex-start

        ：對齊容器最上面(default)

      • flex-end

        ：對齊容器最下面

      • center

        ：垂直置中

      • stretch

        ：將內容元素全部撐開至 Flexbox 的高度

      • baseline

        ：以所有內容元素的基線作為對齊標準

    • align-content

      ：功能同 aling-items，但是是針對多行元素

      • flex-start

        ：對齊容器最上面(default)

      • flex-end

        ：對齊容器最下面

      • center

        ：垂直置中

      • stretch

        ：將內容元素全部撐開至 Flexbox 的高度

      • space-between

        ：平均分配內容元素，左右元素將會與容器的最上面和最下面貼齊

      • space-around

        ：平均分配內容元素，間距也是平均分配

    • flex-wrap

      ：nowrap(default) | wrap
