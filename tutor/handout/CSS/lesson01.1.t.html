簡單選擇器

  1. *

    ：任何元素(萬用字元) 
            
          * { }

  2. tagName

    ：標籤 
            
          <div></div>
        
        
          div { }

  3. #id

    ： 元素的 id 屬性值 
            
          <div id="container"></div>
        
        
          #container { }

  4. .className

    ：元素的 class 屬性值 
            
          <div class="container"></div>
        
        
          .container { }

  5. tagName.className

    ：指定標籤的 class 屬性值 
            
          <div class="container"></div>
        
        
          div.container { }

  6. .className.className

    ：元素的 class 屬性, 同時指定多個屬性值 
              
            <div class="container wrapper"></div>
          
          
            .container.wrapper { }
    註：注意 property 要空白間隔，但 selector 不能空白
