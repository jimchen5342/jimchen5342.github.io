iframe

  1. 用途：框窗；用於在網頁中顯示次網頁
  2. 屬性：
    • src: URL
    • frameborder: 0/1
    • height: px 或 %
    • width: px 或 %
    • sandbox:
      • 空字串: 啟用所有的限制條件
      • allow-same-origin:
      • allow-top-navigation:
      • allow-scripts: 允許執行 javascript
      • allow-forms: 允許表單提交
  3. 事件：onload, onerror, onfocus, onblur, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
