iframe

  1. 用途：框窗；用於在網頁中顯示網頁
  2. 屬性：
    • src: URL
    • frameborder: 0/1
    • height: px 或 %
    • width: px 或 %
    • sandbox:
      • allow-same-origin
      • allow-scripts
      • allow-top-navigation
      • allow-forms
  3. 事件：onload, onerror, onfocus, onblur, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
