h1 ~ h6

  1. 用途：標題
  2. h1：32px 或 2em
  3. h2：24px 或 1.5em
  4. h3：18.72px 或 1.17em
  5. h4：16px 或 1em
  6. h5：13.28px 或 0.83em
  7. h6：10px 或 0.75em
  8. CSS：display: block; font-weight: bold; margin
  9. 屬性：id, class, style
  10. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

p

  1. 用途：段落
  2. CSS：font-size: 16px; display: block; margin: 16px 0;
  3. 屬性：id, class, style
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

div

  1. 用途：區塊
  2. CSS：font-size: 16px; display: block;
  3. 屬性：id, class, style
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

span

  1. 用途：行内元素
  2. CSS：font-size: 16px; display: inline; 預設不能設定高度或寛度
  3. 屬性：id, class, style
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
