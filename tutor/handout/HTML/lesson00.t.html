<!DOCTYPE html>
<html>
  <head>
    <title>基本架構</title>
    <meta charset="utf-8">
    <meta name="viewport" content="minimal-ui, width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">
    <link rel="stylesheet" href="style.css">
    <style type="text/css">
      /**/
    </style>
    <script type="text/javascript" src="system.js"></script>
    <script>
      /**/
    </script>
  </head>	
  <body>
    <!-- 這是備註 -->
  </body>
</html>

標籤(tag)

  1. html
  2. title
  3. meta
    • name="description" content="網頁簡短描述"
    • name="keywords" content="網頁關鍵字"
    • name="author" content="作者姓名"
    • charset="utf-8"
    • name="viewport" content="minimal-ui, width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no"
    • http-equiv="cache-control" content="no-cache"
  4. style：外部，內部，內聯
  5. javascript：外部，內部，內聯
  6. body

通用屬性(property/attribute)

  1. name
  2. id
  3. class
  4. style

注意事項

  1. tag：
    • <tagName>內容</tagName>
    • 自閉合標籤：<br />，<hr />
    不可用自閉合標籤：iframe，textarea，script，canvas
  2. element，node，parent，child，sibling
  3. 大小寫，駝峰式
  4. 單引號，雙引號
  5. 備註：
    • HTML：<!-- -->
    • CSS/JavaScript：/* */
  6. URI/URL：絕對位置，相對位置
    • a.href，link.href
    • img.src，javascript.src，iframe.src
    • form.action
