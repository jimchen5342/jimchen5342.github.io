table

  1. 用途：表格
  2. CSS：display: table;
  3. 屬性
    • border：格線粗細(number; 0: 無框)
    • cellspacing：格外緣距離(number, default: 2)
    • cellpadding：格內緣距離(number)
    • width：寬度(px, %)
  4. 事件：onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

tr

  1. 用途：列；table 的子節點，可包含 th, td 標籤
  2. CSS：display: table-row;
  3. 屬性
    • height: px
    • align
      • left(default)
      • center
      • right
    • valign
      • top
      • middle(default)
      • bottom
  4. 事件：onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

th

  1. 用途：標題格；tr 的子節點
  2. CSS：display: table-cell; font-weight: bold; text-align: center;
  3. 屬性
    • height: px
    • width: px, %
    • align
      • left
      • center(default)
      • right
    • valign
      • top
      • middle(default)
      • bottom
    • colspan：number(欄合併)
    • rowspan：number(列合併)
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

td

  1. 用途：格；tr 的子節點
  2. CSS：display: table-cell;
  3. 屬性
    • height: px
    • width: px, %
    • align
      • left(default)
      • center
      • right
    • valign
      • top
      • middle(default)
      • bottom
    • colspan：number(欄合併)
    • rowspan：number(列合併)
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
