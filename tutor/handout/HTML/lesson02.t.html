ol

  1. 用途：有序清單
  2. CSS：list-style-type: decimal; display: block; margin: 16px 0; padding-left: 40px;
  3. 屬性：id, class, style
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

ul

  1. 用途：無序清單
  2. CSS：list-style-type: disc; display: block; margin: 16px 0; padding-left: 40px;
  3. 屬性：id, class, style
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

li

  1. 用途：ol 或 ul 的子元件
  2. CSS：display: list-item;
  3. 屬性：id, class, style
  4. 事件：onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
