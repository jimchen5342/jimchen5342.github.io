下列 tag 已不建議使用

  1. font
  2. i
  3. b
  4. u
  5. strike

改用下列 CSS

  1. 字體名稱：

    font-family

    : 'Arial Black', 微軟正黑體, Helvetica, Arial;
    註一：字體會因作業系統而有差異
    註二：可一次填寫多個字體，瀏覽器會依序找尋可用的字體
    註三：字體名稱中如有空白，請用引號標註
  2. 字體顏色：

    color:

    blue;
    • 顏色名稱：black, white, red, blue
    • 16進制：#000000, #ffffff, #ff0000, #0000ff
    • rgb：rgb(0, 0, 0), rgb(255, 255, 255), rgb(255, 0, 0), rgb(0, 0, 255)
    • rgba：rgba(0, 0, 0, 0), rgba(255, 255, 255, 0.25), rgba(255, 0, 0, 0.5), rgba(0, 0, 255, 1)
  3. 字體尺寸：

    font-size:

    16px;
    • px
    • em; 1em = 16px
    • pt 已不建議使用
  4. 粗體字：

    font-weight:

    bolder;
    • 100 ~ 900
    • lighter = 100
    • normal = 400
    • bold = 700
    • bolder = 900
  5. 斜體字：

    font-style

    : italic;
  6. 下標線：

    text-decoration

    : underline;
  7. 刪除線：

    text-decoration

    : line-through;
