form

  1. 用途：表單；用於收集各類型的使用者輸入的資訊，回傳給主機
  2. 表單元素：input, textarea, select, option, button
  3. 表單輔助元素：label, fieldset, legend
  4. action：URL, 當表單提交時，向何處發送數據
  5. method：
    • get(default)：網址會帶有 HTML Form 表單的參數與資料；以 ? 開頭，& 為間隔；如：
      https://www.google.com.tw/search?source=hp&q=html5
    • post：
  6. target：
    _self (default)
    _blank
    _parent
    _top
  7. 屬性：id, name
  8. 事件：onreset, onsubmit

input

type='text'

  1. 用途：文字輸入
  2. 屬性：name, id, class, style, size, maxlength, value, placeholder, readonly, disabled
  3. 事件：onkeydonw, onkeyup, onkeypress, onselect, onclick, ondblclick, onfocus, onblur, onchange, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
    4. 				
						<input type='text' id='userid' value='' placeholder='請輸入 ID'/>

type='password'

  1. 用途：密碼輸入
  2. 屬性：name, id, class, style, size, maxlength, value, placeholder, readonly, disabled
  3. 事件：onkeydonw, onkeyup, onkeypress, onselect, onclick, ondblclick, onfocus, onblur, onchange, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
    4. 				
						<input type='password' placeholder='請輸入密碼'/>

type='radio'

  1. 用途：單選圓鈕
  2. 屬性：name, id, class, style, value, checked, disabled
  3. 事件：onclick, onfocus, onblur, onchange, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
    4. 				
						<input type='radio' id='male' name='sex'/>
						<label for='male'>男性</label>
						<input type='radio' id='female' name='sex'/> 
						<label for='male'>女性</label>
    註一：同一個群組 radio，其 name 的值要相同
    註二：建議搭配 label 使用

type='checkbox'

  1. 用途：核取盒
  2. 屬性：name, id, class, style, value, checked, disabled
  3. 事件：onclick, onfocus, onblur, onchange, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout
    4. 				
						<input type='checkbox' id='male'/>
						<label for='male'>男性</label>
    註：建議搭配 label 使用

type='button'

  1. 用途：按鈕
  2. 屬性：name, id, class, style, value, disabled
  3. 事件：onclick, onfocus, onblur, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

註：url, email, tel, number, range, date, time, datetime, search

textarea

  1. 用途：多列文字輸入
  2. CSS：resize
  3. 屬性：name, id, class, style, placeholder, cols, rows, disabled
  4. 事件：onkeydonw, onkeyup, onkeypress, onslect, onclick, onfocus, onblur, onchange, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

select

  1. 用途：選項
  2. 屬性：name, id, class, style, disabled
    • size：顯示 n 列高度，預設為 1
    • multiple：多重選取(當 size 大於 1 時)
  3. 事件：onclick, onfocus, onblur, onchange, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

option

  1. 用途：選項的子元素
  2. 屬性：value, selected
    3. 預設為第一個 option 為 selected
			
					<select size='2' multiple/>
						<option value='male'>男性</option>
						<option value='female' selected>女性</option>
					</select>

button

  1. 用途：按鈕
  2. 屬性：name, id, class, style, disabled
  3. 事件：onclick, onfocus, onblur, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout

fieldset

  1. 用途：分組
  2. 屬性：name, id, class, style

legend

  1. 用途：分組標題
  2. 屬性：name, id, class, style
			
				<fieldset>
					<legend>性別</legend>
					<input type='radio' id='male' name='sex'/>
					<label for='male'>男性</label>
					<input type='radio' id='female' name='sex'/> 
					<label for='male'>女性</label>
				</fieldset>

label

  1. 用途：標簽；主要用於配合 radio, checkbox
  2. for: tagID
  3. 屬性：name, id, class, style
			
				 <input type='checkbox' id='male'/>
				 <label for='male'>男性</label>
