賦值運算子
名稱 簡化的運算子 意義
賦值 x = y x = y
加法賦值 x += y x = x + y
減法賦值 x -= y x = x - y
乘法賦值 x *= y x = x * y
除法賦值 x /= y x = x / y
餘數賦值 x %= y x = x % y

解構賦值：用於提取(extract)陣列或物件中的資料 
		
			// 陣列解構
			var x = [1, 2, 3, 4, 5]; 
			var [y, z] = x; 
			console.log(y); // 1 
			console.log(z); // 2

			// 將剩餘部分解構到一個變數
			var [a, ...b] = [1, 2, 3]; 
			console.log(a); // 1 
			console.log(b); // [2, 3]

			// 物件解構
			var o = {p: 42, q: true}; 
			var {p, q} = o; 
			console.log(p); // 42 
			console.log(q); // true
