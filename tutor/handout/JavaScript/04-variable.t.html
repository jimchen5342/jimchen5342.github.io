1. string：字串(單引號或雙引號)
  2. number：數值(整數及浮點數)
  3. boolean：布林值(true/false)
  4. array：陣列
  5. object：物件
  6. function：函式
  7. null：無值
  8. undefined：未定義或不存在
資料型態物件宣告實字宣告
物件x = new Object()x = {}
字串x = new String()x = "", y = ''
數值x = new Number()x = 1
布林值x = new Boolean()x = true
陣列x = new Array()x = []
日期x = new Date()
函式不建議用function funName(){}
typeof：檢查資料型態 
			
				document.write(typeof "1");              // string
				document.write(typeof '1');              // string
				document.write(typeof 0);                // number
				document.write(typeof true);             // boolean
				document.write(typeof {});               // object
				document.write(typeof (new Object()));   // object
				document.write(typeof (new Date()));     // object
				document.write(typeof (new String(""))); // object
				document.write(typeof [1，2，3]);         // object
				document.write(typeof null);             // object
				document.write(typeof undefined);        // undefined

				function add(x, y){
					return x + y;
				}
				document.write(typeof add);              // function
JavaScript 是一個動態型別的語言，這意味著你不需要在宣告變數時定義它的資料型別，程式執行時會自動轉換
			
				let a = "The answer is: ", b = 25;
				let c = a + b;                   // The answer is: 25
				c = 5;                           // number

				let x = "37" - 7;                // 30
				let y = "37" + 7;                // 377
				let z = parseInt("100", 10) + 7; // 107
数据类型转换
  1. 開頭字元需要是 ASCII 字元(英文大小寫字元)，或是下底線(_)、錢號($)。注意，數字不可用於開頭字元
  2. 接下來的字元可以使用英文字元、數字或下底線(_)
  3. 大小寫是有差異的
  4. 名稱不可使用 JavaScript 語言保留字詞
  5. 名稱被稱為 identifier (識別符，簡稱ID)，它在程式碼上下文中具有唯一性
  6. 駝峰式
			
				let userName = "John";
				let _age = 25;
				let user_id = "john";
