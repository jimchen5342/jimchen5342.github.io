內建函數 說明
escape(string) 傳回 string 的 16 進制編碼字串
unescape(string) 傳回以 16 進制編碼字串的原本字串， 功能與 escape() 剛好相反
eval(string) 把 string 當作 JavaScript 的敘述來執行
isNaN(value) 檢查 value 是否"非數值"型態 (is Not a Number)
parseFloat(string) 傳回 string 轉成浮點小數後的數值
parseInt(string [, base]) 傳回 string 轉成 10 進制後的整數； base 是 string 的進位基數，它是選擇性的參數，有 2、 8、10、16、32，預設為 10。
