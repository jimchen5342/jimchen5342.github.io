內建函式 說明
escape(string) 傳回 string 的 16 進制編碼字串(不建議使用)
不對這些字元作編碼：

ASCII碼 數字 * @ - _ + . /
unescape(string) 傳回解碼 16 進制字串， 功能與 escape() 剛好相反(不建議使用)
encodeURI(string) 將(包含非法字元) URI 字串編碼成符合 URI 格式的新字串
不對這些字元作編碼：

ASCII碼 數字 ? & @ $ # . , : ; + / = - _ ! ~ * ' ( )
decodeURI(string) 傳回解碼字串
encodeURIComponent(string) 與 encodeURI() 相似
不對這些字元作編碼：

ASCII碼 數字 ? & @ $ # , : + / =
decodeURIComponent(string) 傳回解碼字串
eval(string) 把 string 當作 JavaScript 的敘述來執行

eval() 是一個危險的函數
isNaN(value) 傳回檢查 value 是否"非數值"型態 (is Not a Number)
parseFloat(string) 傳回 string 轉成浮點小數後的數值
parseInt(value [, base]) 傳回 value 轉成 10 進制(預設)後的整數；
base 是 value 的進位基數，它是選擇性的參數，有 2、8、10、16、32，預設為 10。
