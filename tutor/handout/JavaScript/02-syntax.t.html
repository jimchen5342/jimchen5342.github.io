語法(syntax)

  1. JavaScript 許多基本語法借鑒自 Java
  2. Case-sensitive（區分大小寫）並使用 Unicode 編碼
  3. 空格、Tab 與換行符號皆被視為空白
  4. 每行指令(instruction)被稱為陳述式(statement)，並用分號（;）分隔； 陳述式會結合運算式，結合的方式可讓它們執行一個完整的工作 
    				
					var a; 
					var b = 2; 
					b++; // 一元運算子
					var c = 2 + 5; // 二元運算子

					var i = 2, j = 3; var k = i * j;

					var x = (a + i ) * k,
						y = (x > 0 ? 1 : 10) * 5;
					
					var z = x > y ? x : y; // 三元運算子

					if(x > y)
						z = x;
					else
						z = y;
