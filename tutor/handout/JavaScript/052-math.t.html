算術運算子
運算子 描述 範例
加號(+) 二元運算子。數字或字串相加 x = 3 + 1;
x = "a" + "b";
x = "x is: " + 25;
一元運算子。將非數值型態運算元轉換成數字。 +"3" 回傳 3
+true 回傳 1
減號(-) 二元運算子。數字相減 x = 5 - 2;
一元運算子。回傳運算元的負數。 假如 x 是 3，-x 回傳 -3。
乘號(*) 二元運算子。回傳兩個運算元相乘後的數。 12 * 5 回傳 60
除號(/) 二元運算子。回傳兩個運算元相除後的數。 12 / 3 回傳 4
取餘數(%) 二元運算子。回傳兩個運算元相除後的餘數。 12 % 5 回傳 2
遞增(++) 一元運算子。將運算元增加 1。
假如使用在運算元之前(++x)，會先回傳增加 1 後的值;
假如使用在運算元之後(x++)，會回傳運算元加 1 前的值。 		假如 x 是 3，
那 ++x 將把 x 設定為 4 並回傳 4，
而 x++ 會回傳 3，接著才把 x 設定為 4。
遞減(--) 一元運算子，將運算元減少 1。
回傳值的情況與遞增運算元相同。		 假如 x 是 3，
那 --x 將把 x 設定為 2 並回傳 2，
而 x-- 會回傳 3，接著才把 x 設定為 2。
指數運算子 (**) 計算以 a 為底的 b 次方，也就是 ab 2 ** 3 回傳 8
10 ** -1 回傳 0.1
