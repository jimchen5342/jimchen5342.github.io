屬性描述
constructor 指定用來建立物件的函式
__proto__ 包含指定之物件內部原型的參考
prototype 傳回物件類別的原型參考
方法描述
hasOwnProperty(key) 返回物件是否有參數 key 指定之名稱的屬性。
靜態方法描述
create(props) 返回一個新物件，指定其原型物件與屬性
assign(target, source...) 返回一個新物件，其所複製一個或多個物件，所有可列舉的屬性。 如果在參數 target 裡的屬性名稱(key)和參數 source 的屬性名稱相同，將會被覆寫。
注意：target 同時也會改變
defineProperty(obj, propName, descriptor) 將屬性加入至物件，或修改現有屬性 (Property) 的屬性值 (Attribute)。

descriptor

  1. enumerable：決定這個屬性能否被 for…in 查找遍歷到，預設為 false

    1. value：屬性預設值，預設為 undefined
    2. configurable：可否變更屬性 (Property) 的屬性值 (Attribute)，及可否刪除屬性 (Property)， 如果為 false 則只能修改 enumerable，預設為 false
    3. writable：可否修改屬性值，預設為 false

    1. set：設定屬性值的函式。它的一個參數包含要指派的值，不可和 value 並傳
    2. get：傳回屬性值的函式；如果未定義 get 存取子，而且嘗試存取屬性值，則會傳回 undefined 值
defineProperties(obj, descriptors) 將一個或多個屬性 (Property) 加入至物件，並且 (或者) 修改現有屬性 (Property) 的屬性 (Attribute)
keys(obj) 返回參數 obj 內所有可列舉屬性的字串陣列
getOwnPropertyNames(obj) 返回可列舉和非可列舉屬性及方法的名稱陣列
getOwnPropertyDescriptor(obj, propName) 返回描述元物件
