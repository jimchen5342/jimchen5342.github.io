屬性描述
title 設定或取得網頁標題
activeElement 傳回目前焦點元件
location 傳回 URL
baseURI 傳回 URL
documentURI 傳回 URL
bgColor 取得或變更網頁背景顏色
fgColor 取得或變更網頁前景顏色
characterSet
charset		 網頁編碼
contentType 網頁內容類型
domain 網域
alinkColor 取得或設定 document 中 a 元件，在滑鼠點擊瞬間顏色值
linkColor 取得或設定 document 中 a 元件連結顏色值
vlinkColor 取得或設定 document 中 a 元件連結過後的顏色值
all 傳回網頁中所有元件的陣列
anchors 傳回標籤 a 元件陣列；元件要有 name 屬性值
links 傳回標籤 a 元件陣列；元件要有 href 屬性值
body 傳回標籤 body 元件
forms 傳回標籤 form 元件陣列
head 傳回標籤 head 元件
images 傳回標籤 images 元件陣列
scripts 傳回標籤 script 元件陣列
styleSheets 傳回標籤 style 元件陣列
function描述
addEventListener(eventName, callback, capture) 註冊事件
  1. eventName：事件名稱，不要加 on
  2. callback：函式變數名稱或匿名函式
    註：callback 傳入參數 Event Object。
    儘量不要用匿名函式。
  3. capture：
    1. false：bubble 方向由元件本身再到父元件；default
    2. true：bubble 方向由父元件再到元件本身
removeEventListener(eventName, callback) 移除事件，callback 不可為匿名函式
dispatchEvent 自訂事件

let event = new Event('build');

// 註冊事件
el.addEventListener('build', function (e) { ... }, false);

// 觸發事件
el.dispatchEvent(event);
createElement(tagName) 依標籤名稱建立 HTML 元件，並傳回；請參考 element 下列方法：
  1. appendChild()
  2. insertBefore()
  3. replaceChild()
  4. insertAdjacentElement()
createTextNode(text) 建立文字節點，並傳回；請參考 element 下列方法：
  1. appendChild()
  2. insertBefore()
  3. replaceChild()
  4. insertAdjacentText()
createAttribute(attrName) 建立屬性節點，並傳回；
請參考 element.setAttributeNode()
getElementById(id) 依 id 查找網頁中的元件，並傳回元件；如找不到則傳回 null
getElementsByClassName(class) 依 className 查找網頁中的元件，並傳回元件陣列
getElementsByName(name) 依 name 查找網頁中的元件，並傳回元件；如找不到則傳回 null
getElementsByTagName(tag) 依 tagName 查找網頁中的元件，並傳回元件陣列
querySelector(selector) 依 selector 查找網頁中的元件，並傳回元件；如找不到則傳回 null
querySelectorAll(selector) 依 selector 查找網頁中的元件，並傳回元件陣列
write(str) 在網頁添加文字或 HTML 元件；避免使用
