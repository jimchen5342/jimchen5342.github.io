屬性描述
id 設定或取得元件在網頁中的唯一識別碼
attributes 傳回元件的屬性陣列
style 設定或取得元件的 style 屬性值集合；style 內的屬性名稱要用駝峰式
classList 傳回元件的 class 屬性值的陣列；
  1. length
  2. contains(value)
  3. add(value)
  4. remove(value)
  5. toggle(value)
className 設定或取得元件的 class 屬性值(字串)
innerHTML 設定或取得元件內的 HTML 內容
outerHTML 設定或取得元件本身及元件內的 HTML 內容
innerText
outerText		 設定或取得元件內的 text 內容
childElementCount 取得子節點的元件數量
clientWidth
clientHeight
傳回元件的寬/高度；不含 margin, border, padding, scrollbar
offsetWidth
offsetHeight
傳回元件的寬/高度；不含 margin
scrollHeight
  1. 在沒有垂直 scrollbar 的情況時；scrollHeight == clientHeight
  2. 在有垂直 scrollbar 的情況時；scrollHeight 才有意義，scrollHeight >= clientHeight
scrollWidth
  1. 在沒有水平 scrollbar 的情況時；scrollWidth == clientWidth
  2. 在有水平 scrollbar 的情況時；scrollWidth 才有意義，scrollWidth >= clientWidth
scrollTop 設定或取得元件頂點與父元件的頂點的距離
scrollLeft 設定或取得元件左邊與父元件左邊的距離
contentEditable 傳回元件是否可編輯
nodeName
tagName		 傳回元件的標籤名稱
parentNode
parentElement		 傳回元件的父節點元件
function描述
addEventListener(eventName, callback, capture) 註冊事件
  1. eventName：事件名稱，不要加 on
  2. callback：函式變數名稱或匿名函式；儘量不要用匿名函式
  3. capture：
    1. false：bubble 方向由元件本身再到父元件；default
    2. true：bubble 方向由父元件再到元件本身
removeEventListener(eventName, callback) 移除事件，callback 不可為匿名函式
appendChild(el) 為元件新增子元件
cloneNode(deep) 複製並傳回元件；
deep(true/false)：是否連元件內的子元件都一併複製
replaceChild(new, old) 代換子元件
removeChild(el) 移除子元件
insertBefore(new, exist) 在某子元件前插入一個新子元件
insertAdjacentElement(position, el) 在指定位置，插入一個新元件。
position：
  1. 'beforebegin'：元件的前面
  2. 'afterbegin'：子節點的第一個位置
  3. 'beforeend'：子節點的最後一個位置
  4. 'afterend':：元件的後面
insertAdjacentHTML(position, html) 把傳入的 html 字串解析成新元件，在指定位置，插入新元件。
position：
  1. 'beforebegin'：元件的前面
  2. 'afterbegin'：子節點的第一個位置
  3. 'beforeend'：子節點的最後一個位置
  4. 'afterend':：元件的後面
insertAdjacentText(position, text) 在指定位置，插入傳入的文字。
position：
  1. 'beforebegin'：元件的前面
  2. 'afterbegin'：子節點的第一個位置
  3. 'beforeend'：子節點的最後一個位置
  4. 'afterend':：元件的後面
contains(otherNode) 在元件的子元件中是否含有 otherNode 元件
hasAttribute(attr) 元件是否有指定屬性，傳回 true/false
hasAttributes() 元件是否具有任何屬性，傳回 true/false
getAttribute(attr) 傳回元件指定屬性的值，如沒有設定則傳回 null
getAttributeNode(attr) 傳回元件指定屬性節點，如沒有設定則傳回 null
setAttribute(attr, value) 設定元件的指定屬性的值
setAttributeNode(attrNode) 設定元件的指定屬性節點
請參考 document.createAttribute();
removeAttribute(attr) 移除元件指定屬性
removeAttributeNode(attr) 移除元件指定屬性節點
hasChildNodes() 元件是否具有任何子元件，傳回 true/false
hasFocus() 元件是否有獲得焦點
querySelector(selector) 依 selector 查找網頁中的元件，並傳回元件；如找不到則傳回 null
querySelectorAll(selector) 依 selector 查找網頁中的元件，並傳回元件陣列
