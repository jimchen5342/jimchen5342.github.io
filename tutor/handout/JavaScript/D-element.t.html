屬性描述
id 元件在網頁中的唯一識別碼
style 設定或取得元件的 style 屬性值
classList 傳回元件的 class 屬性；
  1. length
  2. contains(value)
  3. add(value)
  4. remove(value)
  5. toggle(value)
className 設定或取得元件的 class 屬性值(字串)
innerHTML 設定或取得元件內的 HTML 內容
outerHTML 設定或取得元件內的 HTML 內容，含元件本身
innerText 設定或取得元件內的 text 內容
outerText
childElementCount 取得子節點的元件數量
clientWidth
clientHeight		 傳回元件的寬/高度；不含 margin, border, padding, scrollbar
offsetWidth
offsetHeight		 傳回元件的寬/高度；不含 margin
scrollHeight 在沒有垂直 scrollbar 的情況時, scrollHeight == clientHeight； 在元件的 overflow 樣式值為 hidden，當有 scrollBar 時，scrollHeight 不含 marginTop
scrollWidth 在沒有水平 scrollbar 的情況時, scrollWidth == clientWidth； 在元件的 overflow 樣式值為 hidden，當有 scrollBar 時，scrollHeight 不含 marginTop
scrollTop 設定或取得元件頂點與父元件的頂點的距離
scrollLeft 設定或取得元件左邊與父元件左邊的距離
contentEditable 傳回元件是否可編輯
nodeValue
nodeName
tagName		 傳回元件的標籤名稱
attributes 傳回元件的屬性陣列
childNodes 傳回元件的子節點元件陣列
firstChild 傳回元件的第一個子節點元件
lastChild 傳回元件的最後一個子節點元件
previousSibling
nextSibling
parentNode
parentElement		 傳回元件的父節點元件
function描述
addEventListener(eventName, callback, capture) 註冊事件
  1. eventName：事件名稱，不要加 on
  2. callback：函式變數名稱；儘量不要用匿名函式
  3. capture：
    1. false：bubble 方向由元件本身再到父元件；default
    2. true：bubble 方向由父元件再到元件本身
removeEventListener(eventName, callback) 移除事件，callback 不可為匿名函式
