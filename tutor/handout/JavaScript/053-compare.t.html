比較運算子
運算子 描述 會回傳 true 的例子
等於 (==) 假如運算元相等就回傳 true。 x == 3
s == "3"
3 == '3'
不等於(!=) 假如運算元相等就回傳 true。 x != 4
s != "3"
嚴格等於(===) 假如運算元具有相同型態且相等則回傳 true x === 3
嚴格不等於 (!==) 假如運算元具有相同型態但不相等，或是具有不同型態，回傳 true。 s !== "3"
3 !== '3'
大於 (>) 假如左方運算元大於右方運算元，回傳 true。 x > y
"12" > 2
大於或等於 (>=) 假如左方運算元大於或等於右方運算元，回傳 true。 x >= y
x >= 3
小於 (<) 假如左方運算元小於右方運算元，回傳 true。 x < y
"2" < 12
小於或等於 (<=) 假如左方運算元小於或等於右方運算元，回傳 true。 x <= y
x <= 5
