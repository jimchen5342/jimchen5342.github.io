變數範圍(scope)

  1. 全域(global)
  2. 區域(local)

變數宣告(declare)

  1. var：宣告一個可隨意更改其內容的變數 
    				
					{
						 var a = 0;
					}
					document.write(a);
  2. let：宣告一個可隨意更改其內容的區塊區域變數 
    				
					{
						let a = 0;
						document.write(a);
					}
  3. const：宣告一個只可讀取的不可變常數 
    				
					const PI = 3.1416;
  4. 未宣告 
    				
					a = 0;
    註：會定義全域變數，在嚴格模式裡會產生警告，該定義方式應該盡可能避免

小技巧

  1. 一行陳述式(statement)，多個變數宣告 
    				
					var a = 0, b = 0, c;
  2. 跨行，多個變數宣告 
    				
					var a = 0, 
						b = 0, 
						c = 0;
  3. 'use strict';
