靜態方法描述
isArray(obj) 回傳物件是否為陣列
屬性描述
length 陣列的長度
方法描述
concat(arr [, arr...]) 連接兩個或更多的陣列，並返回新陣列。
toString() 把陣列轉換為字串，並返回字串。
join([str]) 把陣列的所有元素依指定的分隔符號(預設為逗點)進行分隔，放入一個字串並返回字串。
pop() 移除陣列的最後一個元素並回傳
push(elem [, elem...]) 向陣列的末尾添加一個或更多元素，並返回新的長度。
shift() 移除陣列的第一個元素並回傳該元素
unshift(elem [, elem...]) 向陣列的開頭添加一個或更多元素，並返回新的長度。
splice(start, len [, elem...]) 從 start 位置開始移除指定的元素個數，並依需要在這些位置插入新元素。刪除的元素會當做新的 Array 物件傳回。
slice(start [, end]) 返回參數 start 到參數 end(不包括该元素，如沒有則視為到最後一個) 位置元素的新陣列
sort([function]) 對陣列的元素進行排序，如果有提供 function，會依據 function 的傳回值進行排列，否則會轉成字串後排序。
reverse() 對陣列的元素進行反排序。
includes(value) 返回陣列是否包含參數 value 的相同值
indexOf(value) 返回陣列中的元素是否有和參數 value 相同的元素(資料形態也要相同)的位置。否則回傳 -1
fill(value) 陣列所有元素均填入參數 value 的相同值
forEach(callback) 陣列內的每個元素，都執行指定的函式一次。
callback(item [, index, array]);
filter(callback) 返回經指定之函式運算後，由原陣列中通過該函式檢驗之元素所構成的新陣列。
callback(item [, index, array]);
find(callback) 返回一個在陣列中的經回呼函數測試成立的元素值，否則回傳 undefined。。
callback(item [, index, array]);
map(callback) 返回其內容為原陣列的每一個元素經由回呼函式運算後傳的新陣列。
callback(item [, index, array]);
every(callback) 返回經傳入的回呼函示測試所有在陣列中的元素是否全部符合條件。
callback(item [, index, array]);
some(callback) 返回經傳入的回呼函示測試所有在陣列中的元素是否有任一元素符合條件。
callback(item [, index, array]);
reduce(callback) 返回經傳入的回呼函示累加的值。
callback(item [, index, array]);
