屬性描述
length 陣列的長度
方法描述
concat(arr [, arr...]) 連接兩個或更多的陣列，並返回新陣列。
toString() 把陣列轉換為字串，並返回字串。
join([str]) 把陣列的所有元素依指定的分隔符號(預設為逗點)進行分隔，放入一個字串並返回字串。
pop() 移除陣列的最後一個元素並回傳
push(elem [, elem...]) 向陣列的末尾添加一個或更多元素，並返回新的長度。
shift() 移除陣列的第一個元素並回傳該元素
unshift(elem [, elem...]) 向陣列的開頭添加一個或更多元素，並返回新的長度。
splice(start, len [, elem...]) 從 start 位置開始移除指定的元素個數，並依需要在這些位置插入新元素。刪除的元素會當做新的 Array 物件傳回。
slice(start [, end]) 返回參數 start 到參數 end(不包括该元素，如沒有則視為到最後一個) 位置元素的新陣列
sort([function]) 對陣列的元素進行排序，如果有提供 function，會依據 function 的傳回值進行排列。
reverse() 顛倒陣列中元素的順序。
