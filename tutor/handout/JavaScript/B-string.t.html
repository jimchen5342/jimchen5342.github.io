屬性描述
constructor 對創建該物件的函數的引用
length 字串的長度
prototype 向物件添加屬性和方法
靜態方法描述
fromCharCode(n [, n2...]) 依 Unicode 編碼返回字串
方法描述
charAt(n) 返回在指定位置的字元。
charCodeAt(n) 返回在指定的位置的字元的 Unicode 編碼。
concat(s) 連接字串。
indexOf(s [, from]) 返回搜索字串位置，返回 -1 表示找不到，case sensitive。
lastIndexOf(s [, from]) 反向搜索字串，返回 -1 表示找不到，case sensitive。
match(regExp) 返回與正則表達式的匹配的結果，找的到則返回陣列，反之為 null。
replace(regExp, s) 替換與正則表達式匹配的字串，只替換一次。
search(regExp) 返回與正則表達式相匹配的值位置，返回 -1 表示找不到。
slice(start[, end]) 返回依開始位置位到結束位置(未輸入則視為全部)擷取字串的片斷
split([str]) 返回字串依指定字串(預設為逗點)分割為字串陣列。
substr(start[, len]) 返回依開始位置位到指定長度(未輸入則視為全部)擷取字串的片斷。ECMAscript 沒有對該方法進行標準化
substring(start[, end]) 返回依開始位置位到結束位置(未輸入則視為全部)擷取字串的片斷。支援向前擷取子字串
toLowerCase() 把字串全部轉換為小寫。
toUpperCase() 把字串全部轉換為大寫。
repeat(times) 返回原始字串重複指定的次數
startsWith(str[, position]) 返回字串的開頭是否為指定的字串
endsWith(str[, position]) 返回字串的結尾是否為指定的字串
padStart()
padEnd()
trim()
trimLeft()
trimRight()
