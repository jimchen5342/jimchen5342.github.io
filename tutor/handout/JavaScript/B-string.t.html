屬性描述
constructor 對創建該物件的函數的引用
length 字串的長度
prototype 向物件添加屬性和方法
靜態方法描述
fromCharCode(n [, n...]) 依 Unicode 編碼傳回字串
方法描述
charAt(n) 返回在指定位置的字元。
charCodeAt(n) 返回在指定的位置的字元的 Unicode 編碼。
concat(s) 連接字元串。
indexOf(s [, from]) 返回搜索字串位置，返回 -1 表示找不到，case sensitive。
lastIndexOf(s [, from]) 反向搜索字串，返回 -1 表示找不到，case sensitive。
match(regExp) 返回與正則表達式的匹配的結果，找的到則回傳陣列，反之為 null。
replace(regExp, s) 替換與正則表達式匹配的字串。
search(regExp) 返回與正則表達式相匹配的值位置，返回 -1 表示找不到。
slice(start, to) 返回擷取字串的片斷
split() 把字元串分割為字元串數組。
substr(start, len) 從起始索引號提取字元串中指定數目的字元。
substring() 提取字元串中兩個指定的索引號之間的字元。
toLowerCase() 把字元串轉換為小寫。
toUpperCase() 把字元串轉換為大寫。
toString() 返回字元串。

startsWith()
endsWith()
